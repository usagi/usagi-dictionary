## つかいかた // how to use

使いたい辞書ファイル（`.txt`）単位でお使いの IME にインポートしてください。

### [Google日本語入力または互換性のあるIMEの場合] // Google Japanese Input or compatible IME case

「辞書ツール」で .txt ファイルをそのまままインポートするだけです。

### [Microsoft IME または互換性のあるIMEの場合] // Microsoft IME or compatible IME case

Microsoft IME の場合は、.txt ファイル(UTF-8N)を直接インポートできないので、以下の手順で行います。

1. .txt ファイルをメモ帳などで開き、名前を付けて保存で「UTF-16 LE BOM」で保存します。
2. Microsoft IME の「辞書ツール」を開き、「テキストファイルからの登録」を選び、1.で保存したファイルを指定してインポートします。

### [そのたの場合] // Other case

お使いの IME のマニュアルを参照して Google IME 互換の UTF-8N 形式のテキストファイルをインポートする方法をどうにかしてください。
